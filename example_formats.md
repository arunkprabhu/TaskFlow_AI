# Example Meeting Notes - Multiple Formats

## Format 1: Plain Text
Meeting discussed Q2 sprint goals. John will migrate database by Friday. Sarah needs to review PR #123 urgently. Team should update documentation next week. Mike must deploy to production tomorrow.

---

## Format 2: Markdown with Checkboxes
# Sprint Planning Meeting

## Action Items
- [ ] @John: Complete database migration (Due: Friday) - HIGH PRIORITY
- [ ] @Sarah: Review PR #123 - URGENT
- [x] @Mike: Deploy to staging (Completed)
- [ ] Team: Update documentation next week

## Follow-up
TODO: Schedule retrospective meeting for next Tuesday

---

## Format 3: Bullet Points
Sprint Goals:
• Database migration - @John (due Friday, HIGH priority)
• Code review for PR #123 - @Sarah (URGENT)
• Documentation updates - Team (next week)
• Production deployment - @Mike (tomorrow, CRITICAL)
• Performance testing - @Alex (by end of week)

---

## Format 4: Mixed Natural Language
During today's standup meeting on April 22, 2026:

The team discussed the upcoming sprint deliverables. 

Key decisions:
- John committed to completing the database migration by Friday
- Sarah will review PR #123 ASAP (marked as urgent)
- Documentation needs updating according to Mike

ACTION: Mike must deploy the new features to production by tomorrow
TODO: Alex should run performance tests before Friday
NOTE: Schedule a follow-up meeting for next Tuesday

---

## Format 5: Structured List
1. Database Migration
   - Owner: @John
   - Due: Friday (2026-04-26)
   - Priority: High
   - Status: TODO

2. Code Review PR #123
   - Owner: @Sarah  
   - Due: ASAP
   - Priority: Urgent
   - Status: In Progress

3. Deploy to Production
   - Owner: @Mike
   - Due: Tomorrow
   - Priority: Critical
   - Status: TODO

---

## Format 6: Conversational Notes
John: "I'll handle the database migration this week, should be done by Friday"
Sarah: "Let me take care of that urgent PR review today"
Mike: "Need to deploy to production tomorrow morning"
Team Lead: "Everyone should update the docs when you get a chance next week"

TODO Items:
- Performance testing (@Alex - by Friday)
- Security audit (@Team - next sprint)
- Follow-up meeting (next Tuesday)

---

All of these formats will be automatically detected and processed to extract actionable tasks!
